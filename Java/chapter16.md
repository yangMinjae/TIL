# chapter 16

### 1. 인터페이스 (interface)

1.1 인터페이스  
- 객체의 **사용 방법을 정의한 타입**.
- **인터페이스는 추상 메서드의 모음** (추상 클래스가 발전한 형태).
- 특정 규칙을 지킨 **추상 클래스가 인터페이스**이다.
  - **추상 메서드**'만!' 가질 수 있음.
- **작업 지시서 역할**을 수행.

<br>

1.2 인터페이스의 특징 (*추상 클래스와의 차이)  
- **상수**와 **메서드**만 구성 멤버로 가질 수 있다.
- **생성자를 가질 수 없다**.
- **상수는 선언과 동시에 초기값을 지정해야 한다**. (초기값을 지정하지 않으면 에러 발생)

<br>
